## 🏗 Architecture
The app follows the **MVVM (Model-View-ViewModel)** pattern:

1. **Model:** Data entities for `MenuItem` (Room) and JSON response objects.
2. **ViewModel:** Manages UI state and filter logic.
3. **View:** Modularized Compose functions.
