Little Lemon - Meta Android Developer Capstone ProjectWelcome to the Little Lemon mobile application! This project is the final capstone assignment for the Meta Android Developer Professional Certificate. It is a fully functional Android application built from scratch using modern development tools and best practices.🚀 Project OverviewLittle Lemon is a Mediterranean restaurant app that provides a seamless user experience for browsing menu items, filtering by categories, and managing user profiles. The project demonstrates the integration of networking, local databases, state management, and modern UI design.✨ Key FeaturesOnboarding & User Management: A dedicated onboarding flow to collect user details, persisted locally using SharedPreferences.Dynamic Home Screen: Features a "Hero" section with a restaurant description and a real-time Search Bar.Menu Integration: Fetches menu data from a remote API using Ktor Client and stores it in a Room Database for offline access (Single Source of Truth).Category Filtering: Interactive filter buttons (Starters, Mains, Desserts, Drinks) to browse specific sections of the menu.Profile Management: A profile screen where users can view their information or log out to clear their data.Modern Navigation: Implements Jetpack Compose Navigation with back-stack management.🛠 Tech StackLanguage: KotlinUI Framework: Jetpack ComposeArchitecture: MVVM (Model-View-ViewModel)Networking: Ktor Client (CIO Engine & Content Negotiation)Local Persistence: - Room Database (for menu items)SharedPreferences (for user session data)Serialization: Kotlinx SerializationImage Loading: Glide (Compose-optimized)📸 ScreenshotsOnboardingHome ScreenSearch & FilterCategory FilteringProfile Screen🏗 Setup InstructionsClone the repository:Bashgit clone https://github.com/baranDincsoy/little-lemon.git
